Política de Privacidad

Última actualización: 2025-05-21

Bienvenido/a a la Política de Privacidad de la aplicación "PROGRAMACIÓN DE ENSAYOS" (en adelante, "la Aplicación"). Su privacidad es importante para nosotros. Esta política explica cómo recopilamos, usamos, divulgamos y protegemos su información personal cuando utiliza nuestra Aplicación, incluyendo cualquier dato obtenido a través de las APIs de Google.

1. Información que Recopilamos

Podemos recopilar y procesar la siguiente información sobre usted:

  • Información proporcionada por usted: Esto incluye la información que proporciona al registrarse para usar la Aplicación, al completar formularios, o al comunicarse con nosotros.
  • Información de Google APIs: Si autoriza a la Aplicación a acceder a su cuenta de Google (por ejemplo, Google Calendar, Google Sheets), recopilaremos la información necesaria para proporcionar las funcionalidades de la Aplicación. Esto puede incluir [listar específicamente qué datos se leen/escriben, por ejemplo: eventos de calendario, datos de hojas de cálculo específicas]. El uso de la información recibida de las APIs de Google se adherirá a la Política de Datos de Usuario de los Servicios API de Google, incluyendo los Requisitos de Uso Limitado.
  • Información de uso: Detalles de su uso de la Aplicación, incluyendo, pero no limitado a, datos de tráfico, datos de localización, weblogs y otros datos de comunicación.

2. Cómo Usamos Su Información

Utilizamos la información que tenemos sobre usted de las siguientes maneras:

  • Para proporcionarle y mejorar la Aplicación y sus funcionalidades.
  • Para gestionar su cuenta y proporcionarle soporte al cliente.
  • Para cumplir con nuestras obligaciones legales y regulatorias.
  • Uso de datos de Google APIs: La información obtenida a través de las APIs de Google se utilizará exclusivamente para [describir el propósito específico, por ejemplo: programar ensayos en su calendario, leer datos de configuración de una hoja de cálculo]. No utilizaremos estos datos para publicidad ni los transferiremos a terceros, excepto cuando sea necesario para cumplir con la ley aplicable o como parte de una fusión, adquisición o venta de activos.

3. Requisitos de Uso Limitado (Google API Services User Data Policy)

El uso y la transferencia por parte de la Aplicación de información recibida de las APIs de Google se adherirán a la Política de Datos de Usuario de los Servicios API de Google, incluyendo los Requisitos de Uso Limitado.

Esto significa que:

  • Solo usaremos el acceso para leer, escribir, modificar o controlar los cuerpos de los mensajes de Gmail (incluidos los archivos adjuntos), metadatos, encabezados y configuraciones para proporcionar un cliente de correo electrónico web que permita a los usuarios redactar, enviar, leer y procesar sus correos electrónicos y no transferiremos estos datos de Gmail a otros a menos que hacerlo sea necesario para proporcionar y mejorar estas funciones, cumplir con la ley aplicable o como parte de una fusión, adquisición o venta de activos.
  • No utilizaremos estos datos de Gmail para publicar anuncios.
  • No permitiremos que los humanos lean estos datos a menos que tengamos su consentimiento afirmativo para mensajes específicos, lo necesitemos por motivos de seguridad, como investigar abusos, para cumplir con la ley aplicable, o para las operaciones internas de la Aplicación y aun así solo cuando los datos hayan sido agregados y anonimizados.

Nota del desarrollador: Adapte la sección anterior específicamente a los permisos de Google API que su aplicación utiliza. Si no usa Gmail API, elimine o modifique esa sección. Sea específico sobre CADA API de Google que utiliza y cómo cumple con los requisitos de uso limitado para esa API.

4. Divulgación de Su Información

No compartiremos su información personal con terceros, excepto en las siguientes circunstancias:

  • Con su consentimiento explícito.
  • Si estamos obligados a divulgar o compartir sus datos personales para cumplir con cualquier obligación legal.
  • Para proteger los derechos, la propiedad o la seguridad de la Aplicación, nuestros usuarios u otros.

5. Seguridad de Sus Datos

Tomamos medidas razonables para proteger su información personal contra pérdida, robo, uso indebido y acceso no autorizado, divulgación, alteración y destrucción. Sin embargo, ninguna transmisión por Internet o sistema de almacenamiento electrónico es completamente seguro.

6. Sus Derechos

Usted tiene derecho a acceder a la información que tenemos sobre usted, solicitar correcciones y, en algunas circunstancias, solicitar la eliminación de sus datos personales. Para ejercer estos derechos, contáctenos en calidadsuelosyp@gmail.com, direcciontecsyp@gmail.com o metrologiasuelosyp@gmail.com.
